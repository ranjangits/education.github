HTML Basic Examples

In this chapter we will show some basic HTML examples.
Don't worry if we use tags you have not learned about yet.


HTML Documents

All HTML documents must start with a document type declaration: !DOCTYPE html. The HTML document itself begins withhtml and ends with /html . The visible part of the HTML document is betweenbody and /body .

Example

< !DOCTYPE   html >

<html >

< head >

</ head>

</ head>

< body >

< h1 > My First Heading </h1 >

< P > My First paragraphs </P >

</ body >

</html >



The < !DOCTYPE> Declaration


The !DOCTYPE declaration represents the document type, and helps browsers to display web pages correctly.
It must only appear once, at the top of the page (before any HTML tags).
